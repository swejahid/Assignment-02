# Assignment-02
Live Demon Link : https://swejahid.github.io/Assignment-02/
A responsive JavaScript project that uses the CocktailDB API to search and display drinks.

## Features

- Load default drinks from API
- Search drinks by name
- Add drinks to Selected Group
- Maximum 7 selected drinks
- Prevent duplicate drinks
- Bootstrap Details Modal
- Responsive Design
- Modern UI

## Technologies

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6)
- Fetch API
- CocktailDB API

## API

https://www.thecocktaildb.com/api.php
